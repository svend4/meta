## 1. Общая структура книги

Книга описывает **«Тотальную систему боя»** (ТСБ) — авторскую систему рукопашного боя,
построенную на строгих математических и биомеханических принципах.

Автор строит систему снизу вверх: от базовых движений (восьмёрка)
до полной классификации боевых ситуаций и стратегий.

## 2. Ключевые математические концепции

### 2.1 Восьмёрка — фундаментальная траектория

Базовое движение системы — **лемниската** (∞):

```
Параметрические уравнения (лемниската Бернулли):
  x(t) = r · sin(t)
  y(t) = r · sin(t) · cos(t)

Или в форме Лиссажу:
  x(t) = a · sin(2t)
  y(t) = b · sin(t)
```

Свойства восьмёрки в ТСБ:
- **Самозамкнутость** — бесконечный цикл без точки остановки
- **Инь-ян** — две петли = две полярные половины (рис. 9.7)
- **Экономия** — минимальный путь между двумя атакующими зонами
- **Кривизна** — всегда кривая, никогда прямая (закон кривизны)

### 2.2 Система сфер МВС/СВС/БВС

Три концентрические боевые зоны вокруг тела:

```
МВС (Малая Внутренняя Сфера):    r₁ ≈ 30 см  — контактная зона
СВС (Средняя Внутренняя Сфера):  r₂ ≈ 60 см  — ближний бой
БВС (Большая Внешняя Сфера):     r₃ ≈ max    — дистанционный бой

Вложенность: МВС ⊂ СВС ⊂ БВС
```

Каждая сфера разбита **восьмёркой** на две полусферы (инь/ян):

```
МВС: верх-низ восьмёркой → 2 полусферы
СВС: лево-право восьмёркой → 2 полусферы
БВС: перед-зад восьмёркой → 2 полусферы
```

### 2.3 Система окон — **прямая связь с Q6**

**Центральная структура системы.** Тело разбито на **6 окон** (зон):

```
Верхнее левое  (ВЛ) — бит 0
Верхнее правое (ВП) — бит 1
Среднее левое  (СЛ) — бит 2
Среднее правое (СП) — бит 3
Нижнее левое   (НЛ) — бит 4
Нижнее правое  (НП) — бит 5

Каждое окно: ОТКРЫТО = 1, ЗАКРЫТО = 0

h = bit5·bit4·bit3·bit2·bit1·bit0  ∈ {0..63} = Q6
```

**64 возможных боевых состояния тела** — в точности вершины Q6!

Примеры:
```
h = 000000 (0)  — все закрыто  → полная защита
h = 111111 (63) — всё открыто → максимальная атака / уязвимость
h = 010101 (21) — правые окна → атака вправо
h = 101010 (42) — левые окна  → атака влево
h = 000111 (7)  — нижние+среднее право → удар ногой + защита рукой

Инь-ян в Q6:
  yang(h) = число открытых окон (0..6)
  h и antipode(h) = 63-h: h ↔ полная инверсия (закрытое↔открытое)
```

### 2.4 Законы чёрного ящика

Алгоритмические законы для построения техник:

| Закон | Формулировка | Q6-интерпретация |
|-------|-------------|------------------|
| **Закон 1** | ≤ 2 движений для поражения 1 цели | Гамминг-расстояние ≤ 2 в Q6 |
| **Закон 2** | ≤ 3 движений в одной атаке | Путь длины ≤ 3 в Q6 |
| **Закон памяти** | Сумма движений техники ≤ 9 | Длина пути в Q6 ≤ 9 |
| **Оптимум** | 7-9 техник в арсенале | 7-9 орбит в Q6 |
| **Закон 3:1** | 3 движения руками на 1 шаг | Q3(руки) × Q1(ноги) |

### 2.5 Законы нечётных действий

```
Закон нечётных движений:
  При нечётном числе движений возможно зеркальное повторение
  без прерывания ритма.

Закон нечётных шагов:
  При нечётном числе шагов возможна смена направления при вращении.

Закон симметрии:
  Симметрия определяется нечётным числом фрагментов.
```

Q6-интерпретация:
```
Чётный путь в Q6: возвращает к той же паритетности ян
Нечётный путь в Q6: меняет паритетность ян
"Зеркальное повторение" = симметрия относительно антипода
```

### 2.6 Закон сильного справа — круговая доминанта

```
Закон: в круговом противостоянии А, Б, В:
  А побеждает Б, Б побеждает В, В побеждает А
  (нетранзитивное отношение — как камень-ножницы-бумага)

Расширение: круг из 8-9 боевых стилей животных
  (рис. 13.4: тигр, леопард, журавль, змея, богомол, обезьяна...)
  Каждый сильнее предыдущего и слабее следующего.
```

Q6-интерпретация: граф доминанты = ориентированный цикл = Ham-цикл в Q6.

### 2.7 Классификация режимов действий

| Режим | Описание | Характер |
|-------|---------|---------|
| Фехтование | Дистанционные укалывания | БВС |
| Строительство этажей | Послойная атака зон | МВС↔СВС |
| Пугливая обезьяна | Уклоны + контратака | МВС |
| Удары богомола | Серийные удары руками | СВС |
| Вьетнамский ключ | Захваты + рычаги | МВС |

5 режимов × 3 сферы = 15 тактических конфигураций.

## 3. Структурный анализ

### 3.1 Иерархия системы

```
Уровень 0: ВОСЬМЁРКА (базовое движение)
    ↓
Уровень 1: ПЕТЛИ (замкнутые траектории)
    ↓
Уровень 2: ОКНА (6 бинарных зон = Q6)
    ↓
Уровень 3: СФЕРЫ (МВС/СВС/БВС)
    ↓
Уровень 4: РЕЖИМЫ ДЕЙСТВИЙ (5 режимов)
    ↓
Уровень 5: ЯЗЫК БОЯ (стратегическое мышление)
```

### 3.2 Связь восьмёрки с Q6

```
Восьмёрка = лемниската = граф θ (два цикла, общая вершина)
Q6 = 6-мерный гиперкуб

Маршрут восьмёрки по Q6:
  0 → 1 → 3 → 2 → 0   (первая петля, bits 0,1)
  0 → 4 → 5 → 1 → 0   (пересечение через 0)
  ...

Инь-ян восьмёрки → бинарное деление Q6:
  Чётный yang: {0,2,4,...} — «инь»-половина
  Нечётный yang: {1,3,5,...} — «ян»-половина
```

## 5. Уникальность

Система Крюкова — единственная известная боевая система,
**явно построенная на бинарной классификации 6 зон тела**
(= Q6 = И-цзин без явной отсылки).

Аналогия с И-цзин непреднамеренная, но структурно точная:
- 6 битов (черты гексаграммы) → 64 состояния
- Инь/ян каждой черты → открыто/закрыто каждого окна
- Закон нечётных действий ↔ законы гексаграмм
