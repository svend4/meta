# hexcore

Общее ядро всех hex-проектов. Реализует граф Q6 — математическую основу системы гексаграмм.

## Что это

64 гексаграммы образуют **граф Q6** (6-мерный гиперкуб):
- каждый узел = гексаграмма (6-битное число 0..63)
- каждое ребро = изменение одной черты (расстояние Хэмминга = 1)
- каждый узел имеет ровно **6 соседей**

Это граф Кэли группы (Z₂)⁶. Те же структуры используются в:
теории кодирования (коды Хэмминга), квантовых вычислениях (ворота Паули-X),
синтезе цифровых схем (карты Карно), теории автоматов.

## API

```python
from libs.hexcore.hexcore import (
    neighbors,      # 6 соседей гексаграммы
    hamming,        # расстояние Хэмминга
    flip,           # перевернуть одну черту
    shortest_path,  # BFS кратчайший путь
    all_paths,      # все пути (с ограничением длины)
    gray_code,      # гамильтонов путь через все 64 узла
    antipode,       # противоположная гексаграмма
    describe,       # словарь характеристик
    render,         # текстовый рендер
)
```

## Запуск демо

```bash
python3 libs/hexcore/hexcore.py
```
