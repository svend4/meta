## Ключевые тождества

```
64 = 4³ = 2⁶ = 8²

КУБ ↔ СФЕРА:
  Вписанная сфера: r = a/2
  Описанная сфера: R = a√3/2
  R/r = √3  (отношение сфер = диагональ единичного куба)

ДИАГОНАЛЬ КУБА → ШЕСТИУГОЛЬНИК:
  Проекция куба вдоль главной диагонали (1,1,1)/√3
  → правильный шестиугольник (6 рёбер, 6 вершин)
  → центральная точка = сама диагональ = ось симметрии
  → шестиугольник = гайка/звезда Давида с точкой в центре

ДВА КУБА = ВОСЬМЁРКА:
  Восьмёрка (лемниската) = два шара/куба, соединённых вместе
  Q6 × Q6 = Q12 = «гантели» = две руки / два объекта
```

## Проект 1: `hexliuxing` — Шесть элементов (расширение У-Синь)

**Идея:** У-Синь (5 элементов, пятиугольник) → 6 элементов (шестиугольник).
**Связь с Q6:** 6 элементов = 6 бит = 6 измерений Q6.

### Шестиугольник элементов

```
        Элемент 0 (Красный/R)
       /                   \
Эл.5 (Пурп.)           Эл.1 (Жёлт.)
  |          ☯ центр       |
Эл.4 (Синий)           Эл.2 (Зелён.)
       \                   /
        Элемент 3 (Голубой/C)
```

Расстояния на шестиугольнике (аналог У-Синь):

```
1 шаг  = ТВОРЕНИЕ   (сосед → порождает следующий)
2 шага = СОХРАНЕНИЕ (через один → поддерживает)
3 шага = РАЗРУШЕНИЕ (противоположный = дополнительный цвет)
4 шага = УГАСАНИЕ   (= 2 шага назад)
5 шагов = ДЕГРАДАЦИЯ (= 1 шаг назад)
6 шагов = ОБНОВЛЕНИЕ (возврат = рождение заново)
```

Сравнение с У-Синь:
```
У-Синь (5 элементов, пятиугольник):
  Цикл творения:    1 шаг по кругу
  Цикл разрушения:  2 шага через один

Лю-Синь (6 элементов, шестиугольник):
  Творение:   1 шаг
  Сохранение: 2 шага       ← НОВЫЙ цикл!
  Разрушение: 3 шага (диаметр = дополнительный цвет)
```

### Цвета = элементы (аддитивная и субтрактивная модели)

```
Позиция  Цвет      Дополн.   Биты RGB
   0     Красный   Голубой   100/011
   1     Жёлтый    Синий     110/001
   2     Зелёный   Пурпурн.  010/101
   3     Голубой   Красный   011/100
   4     Синий     Жёлтый    001/110
   5     Пурпурн.  Зелёный   101/010

Дополнительные пары = диаметры шестиугольника = расстояние 3
  (противоположные на шестиугольнике)
Смешение соседей (1 шаг) → средний цвет:
  Красный + Жёлтый → Оранжевый
  Жёлтый + Зелёный → Жёлто-зелёный
  и т.д.
```

### Звезда Давида = Лю-Синь в Q6

```
Шестиугольник + все диагонали = звезда Давида

Два треугольника звезды:
  △₁ (чётные): эл. 0, 2, 4  (R, G, B — аддитивные первичные)
  △₂ (нечётные): эл. 1, 3, 5 (Y, C, M — субтрактивные первичные)

В Q6:
  △₁ = вершины с yang % 3 == 0 (в определённой кодировке)
  △₂ = вершины с yang % 3 == 1
```

## Проект 2: `hexcubsphere` — Кубатура сферы / Шаратура куба

**Идея:** Формализовать отношения между кубом и его сферами.

### Основные формулы

```
Куб со стороной a:

1. Вписанная сфера (касается центров граней):
   r = a/2
   V_шар = (4/3)π(a/2)³ = πa³/6
   V_куб / V_шар = 6/π ≈ 1.910

2. Описанная сфера (проходит через вершины):
   R = a√3/2
   V_шар = (4/3)π(a√3/2)³ = π√3·a³/2
   V_куб / V_шар = 2/(π√3) ≈ 0.368

3. Средняя сфера (проходит через середины рёбер):
   ρ = a√2/2
   V_куб / V_шар = 3/(π√2) ≈ 0.674

4. «Кубатура шара»: куб эквивалентного объёма
   Шар радиуса R → куб со стороной a:
   a = (4πR³/3)^(1/3) = R·(4π/3)^(1/3) ≈ 1.612·R

5. «Шаратура куба»: шар эквивалентного объёма
   Куб со стороной a → шар радиуса R:
   R = (3a³/4π)^(1/3) = a·(3/4π)^(1/3) ≈ 0.620·a
```

### Связь с Q6 (дискретная «шаратура»)

```
Шар Хэмминга радиуса r в Q6 (центр h=0):
  |ball(h, r)| = Σ C(6,k) для k=0..r

  r=0: C(6,0) = 1      (только центр)
  r=1: 1+6 = 7         (МВС Крюкова)
  r=2: 7+15 = 22       (СВС Крюкова)
  r=3: 22+20 = 42      (БВС Крюкова)
  r=6: 64              (весь Q6)

«Радиус» Q6 = 6 (диаметр между антиподами)
«Вписанная сфера» Q6 = r=1: 7 вершин
«Описанная сфера» Q6 = r=6: 64 вершины
Аналог R/r = 6/1 = 6
```

## Проект 3: `hextwoq6` — Два куба / Гантели / Q6 × Q6

**Идея:** Восьмёрка = два куба/шара, соединённых вместе = Q12.
**Применение:** Две руки человека, два объекта, робот-манипулятор.

```
Q12 = Q6 ⊗ Q6:
  128 вершин = 64×64 пар состояний
  Состояние системы: (h₁, h₂) где h₁, h₂ ∈ Q6

Для двух рук по Крюкову:
  h₁ = состояние правой руки (BodyState)
  h₂ = состояние левой руки  (BodyState)
  (h₁, h₂) ∈ Q6 × Q6 = 4096 пар

Восьмёрка как переход:
  (h₁, h₂) → ... → (h₂, h₁)  (обмен ролей рук)
  Это маршрут в Q6 × Q6
```

## Диагональная проекция куба

```
Куб (0,0,0)→(1,1,1), смотрим вдоль (1,1,1)/√3:

Видим:
  Вершина (0,0,0): центральная точка (дальняя)
  Вершина (1,1,1): центральная точка (ближняя)
  6 промежуточных вершин: 6 точек шестиугольника

Шестиугольник:
  Три вершины yang=1: (1,0,0), (0,1,0), (0,0,1)
  Три вершины yang=2: (1,1,0), (1,0,1), (0,1,1)

Это: шестерня / гайка / звезда Давида с точкой в центре
     (два треугольника = yang=1 и yang=2)

Q6-аналог: проекция вдоль диагонали 0→63
  Слои по yang: 1, 6, 15, 20, 15, 6, 1
  Центральный слой (yang=3): 20 вершин = «экватор»
```
