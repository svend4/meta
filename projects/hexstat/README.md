# hexstat — Статистика и теория информации на Q6

Вероятностные распределения на Q6 = (Z₂)⁶, энтропия, дивергенции,
случайные блуждания, статистические тесты, взаимная информация.

## Статус

Реализован

## Концепция

```
Q6Distribution: вероятностное распределение p: {0..63} → [0,1]
  Энтропия Шеннона: H(p) = −Σ pᵢ log₂ pᵢ
  Максимум: H = 6 бит (равномерное распределение)

Стационарное блуждание на Q6: РАВНОМЕРНОЕ (граф регулярен)
```

## Ключевые возможности

- **Q6Distribution**: энтропия, KL-дивергенция, TV-расстояние, выборка
- **Моменты**: E[yang_count], маргинальные распределения, корреляции битов
- **RandomWalk**: случайное блуждание по Q6, время смешивания
- **Тесты**: χ²-тест, KS-тест (проекция на yang_count)
- **Взаимная информация**: I(X;Y) для пар битов гексаграмм
- **Максимально-энтропийное**: MaxEnt при заданном E[yang]
- **Ёмкость BSC**: двоичный симметричный канал через Q6

## Информационные характеристики Q6

```
Равномерное:    H = log₂(64) = 6 бит
Диам. Q6:       6 (= число бит)
BSC ёмкость:    C(p) = 1 − H_b(p) бит (через H_b — двоичная энтропия)
```
