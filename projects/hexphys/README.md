# hexphys — Статистическая физика на Q6

6 спинов ∈ {−1, +1} — конфигурации Q6. Модель Изинга с точным решением,
газ Янга, алгоритм Метрополиса, квантовые операции над 6 кубитами.

## Статус

Реализован

## Концепция

```
Гексаграмма h = (σ₀,...,σ₅) ∈ {0,1}⁶ — конфигурация 6 спинов:
  σᵢ = 2·bitᵢ − 1 ∈ {−1, +1}   (инь = −1, янь = +1)

Сумма по состояниям = ТОЧНО 64 слагаемых (весь Q6)
```

## Реализованные модели

| Модель | Описание |
|--------|----------|
| `IsingChain` | Цепочка Изинга: E = −J Σσᵢσᵢ₊₁ − B Σσᵢ |
| `YangGas` | Газ Янга: ян-число = число частиц, μ = хим. потенциал |
| `MetropolisQ6` | Алгоритм Метрополиса: флип случайного бита |
| `QuantumBits` | 6-кубитное состояние: вектор 64 амплитуды |

## Точное решение Изинга

```
Z(β) = Σ_{h∈Q6} exp(−β E(h))   — 64 слагаемых, точно!
Трансфер-матрица T (2×2):
  T[s][s'] = exp(β(J·s·s' + B·(s+s')/2))
  Z = Tr(T^6) для периодических граничных условий
```

## Физические наблюдаемые

- Свободная энергия: F = −(1/β) ln Z
- Намагниченность: M = ⟨Σσᵢ⟩ / 6
- Теплоёмкость: C = β² (⟨E²⟩ − ⟨E⟩²)
- Восприимчивость: χ = β (⟨M²⟩ − ⟨M⟩²) · 6
