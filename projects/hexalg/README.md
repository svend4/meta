# hexalg — Гармонический анализ на Q6

Гармонический анализ на абелевой группе Q6 = (Z₂)⁶: характеры, преобразование
Уолша–Адамара, свёртка, bent-функции.

## Статус

Реализован

## Концепция

```
Q6 как абелева группа (Z₂)⁶:
  64 характера: χ_u(h) = (−1)^{⟨u,h⟩}
  WHT = дискретное преобразование Фурье на Q6
  Свёртка: (f*g)(h) = Σ_a f(a)·g(a⊕h)
  Теорема о свёртке: F(f*g) = F(f)·F(g) (поточечно)
  Тождество Парсеваля: Σ|f|² = (1/64)Σ|F(f)|²
```

## Математическая база

- Характеры (Z₂)⁶: χ_u(h) = (−1)^{popcount(u & h)}
- WHT (Walsh–Hadamard Transform): butterfly O(n log n)
- Bent-функции: |WHT(f)(u)| = 8 для всех u (максимальная нелинейность)
- Граф Кэли Cay(Q6, S): собственные значения через WHT
- Спектр гиперкуба: λ_k = 6−2k, кратность C(6,k)
- Групповое кольцо F₂[Q6]: свёртка над GF(2)
