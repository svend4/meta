# hexdim — Q6 как 6-мерный гиперкуб

Исследование размерной структуры Q6 = (Z₂)⁶: подкубы, проекции, разложения,
код Грея, псевдо-QR-представление.

## Статус

Реализован

## Концепция

```
Q6 = шестимерный двоичный гиперкуб:
  64 вершины = 6-битные числа 0..63
  192 ребра  = пары с расстоянием Хэмминга 1
  Диаметр    = 6

Подкубы Qₖ внутри Q6:
  Q0: C(6,0)=1   — точка
  Q1: C(6,1)=6   — ребро
  Q2: C(6,2)=15  — квадрат
  Q3: C(6,3)=20  — куб (триграмма порождает куб!)
  Q4: C(6,4)=15  — тессеракт
  Q5: C(6,5)=6   — пятимерный куб
```

## Математическая база

- Разложения: Q6 = Q3×Q3 (нижняя × верхняя триграмма)
- Код Грея: гамильтонов путь, упорядочивающий 64 гексаграммы
- Псевдо-QR: 6 линий = 6 бит = адрес в Q6
- Проекции Q6 → R² (8×8 сетка), Q6 → R³ (3D-вложение)
- Расширение: 4 состояния/линия → Q12 = 4⁶ = 4096 = 2¹²

## Структура И-Цзин

```
Q6 = Q3(нижняя) × Q3(верхняя)
  нижняя триграмма: биты 0,1,2  (8 триграмм)
  верхняя триграмма: биты 3,4,5 (8 триграмм)
  64 = 8 × 8 гексаграммы
```
