# hexlearn — Машинное обучение на Q6

Алгоритмы машинного обучения на дискретном метрическом пространстве
Q6 = (Z₂)⁶ с метрикой Хэмминга. Только stdlib Python, без numpy/scikit-learn.

## Статус

Реализован

## Концепция

```
Q6 со метрикой Хэмминга — дискретное метрическое пространство.
Задача: (hexagram, label) → классификатор / кластеризатор.
Нет вещественных координат — только расстояния d(a, b) = popcount(a ⊕ b).
```

## Реализованные модели

| Модель | Описание |
|--------|----------|
| `KNN` | k ближайших соседей по метрике Хэмминга |
| `KMedoids` | k-медоидов (аналог k-means в метрическом пространстве) |
| `SpectralEmbed` | Спектральное вложение Q6 → R^d через лапласиан |
| `MarkovChain` | Случайное блуждание / Марковская цепь на Q6 |
| `HammingBayes` | Наивный Байес по битам гексаграммы |

## Математическая база

- Медоид: argmin_{m ∈ S} Σ d(m, x) — существует всегда в конечном пространстве
- Лапласиан Q6: L = D − A, собственные векторы = WHT-базис
- Стационарное распределение случайного блуждания: равномерное (Q6 регулярен)
- Марковское смешивание: O(n log n) = O(6 log 6) ≈ 15 шагов
