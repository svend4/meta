# hexsym — Группа автоморфизмов Q6

Aut(Q6) = гиперoctaэдральная группа B₆ = S₆ ⋉ (Z₂)⁶ порядка 46 080.
Орбиты, стабилизаторы, лемма Бёрнсайда, подгруппы.

## Статус

Реализован

## Концепция

```
Aut(Q6) = B₆ = S₆ ⋉ (Z₂)⁶

  |Aut(Q6)| = 6! × 2⁶ = 720 × 64 = 46 080

Каждый автоморфизм: (π, m)
  π ∈ S₆   — перестановка 6 позиций битов
  m ∈ Q6   — маска XOR (глобальный «переворот»)
  Действие: h → π(h) ⊕ m

Умножение: (π₁,m₁)·(π₂,m₂) = (π₁∘π₂, π₁(m₂) ⊕ m₁)
```

## Структура орбит

```
Под Aut(Q6): 1 орбита (Q6 вершинно-транзитивен)
Под S₆:     7 орбит по весу Хэмминга (= yang_count)
  yang=0: {0}         — 1 гексаграмма
  yang=1: 6 штук
  yang=2: 15 штук
  yang=3: 20 штук
  yang=4: 15 штук
  yang=5: 6 штук
  yang=6: {63}        — 1 гексаграмма
```

## Генераторы Aut(Q6)

```
τᵢ = транспозиция битов i и i+1, i=0..4  (5 генераторов S₆)
σ  = переворот бита 0                      (генератор (Z₂)⁶ части)
→ 6 генераторов порождают всю Aut(Q6)
```
