# hexnav — Интерактивный навигатор Q6

Базовый инструмент экосистемы. Визуальный интерфейс для навигации по графу Q6:
выбрать текущую гексаграмму, увидеть 6 соседей, записать путь.

## Статус

Идея → в разработке

## Что должно делать

- Показывать текущую гексаграмму (текстовый/ASCII рендер)
- Показывать 6 возможных переходов (соседей)
- Записывать путь (последовательность гексаграмм) как «программу»
- Экспортировать путь: бинарная строка / JSON / текст
- Режим поиска: найти кратчайший путь между двумя гексаграммами

## Почему это первый проект

Это «оболочка» поверх hexcore. Как только он готов — служит базой для
тестирования всех остальных проектов экосистемы.

## Стек

Python 3.10+ (CLI). Опционально: веб-интерфейс (HTML + vanilla JS, без фреймворков).

## Зависимости

- `libs/hexcore` (обязательно)

## Запуск (будущий)

```bash
python3 projects/hexnav/hexnav.py
```
