# hexgf — Поле Галуа GF(2⁶) на 64 гексаграммах

GF(2⁶) — конечное поле из 64 элементов, изоморфное Q6 как аддитивной группе.
Умножение определяется примитивным неприводимым многочленом p(x) = x⁶ + x + 1.

## Статус

Реализован

## Концепция

```
GF(2^6) = GF(2)[x] / (x^6 + x + 1)

Каждый элемент ↔ 6-битное число 0..63:
  bit k = коэффициент при x^k

Сложение:   a ⊕ b   (XOR — совпадает с Q6!)
Умножение:  беспереносное + редукция mod p(x)
Обратный:   a^{-1} = a^{61} (теорема Ферма: a^{63}=1)
Примитивный: g = 2 = x, ord(g) = 63
```

## Ключевые структуры

- **Таблицы exp/log**: быстрое умножение через логарифмы
- **Абсолютный след**: Tr(a) = a ⊕ a² ⊕ a⁴ ⊕ a⁸ ⊕ a¹⁶ ⊕ a³² ∈ {0,1}
- **Циклотомические классы**: {a, a², a⁴, ...} → минимальные многочлены
- **Подполя**: GF(2) ≤ GF(4) ≤ GF(8) ≤ GF(64)
- **Логарифмы Цеха**: Z[k] = log(1 + g^k)
- **Аддитивные характеры**: ψ_b(a) = (−1)^{Tr(b·a)}

## Математическая база

- |GF(2^6)*| = 63 (мультипликативная группа — циклическая)
- Минимальный многочлен: минимальный многочлен элемента над GF(2)
- BCH-коды: задаются нулями в GF(2^6)
