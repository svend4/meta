# hexgraph — Граф-теоретический анализ Q6

Q6 как граф: 64 вершины, 192 ребра. Подграфы, независимые множества,
хроматическое число, гамильтоновы пути, спектральный анализ.

## Статус

Реализован

## Концепция

```
Граф Q6:
  Вершины:        64 гексаграммы (0..63)
  Рёбра:          192 (каждая вершина — степень 6)
  Диаметр:        6
  Хроматическое: 2 (двудольный: чётный/нечётный вес)
  Гамильтонов:   да (содержит гамильтонов цикл)
```

## Ключевые возможности

- **Подграфы**: создание произвольных подграфов Q6
- **Независимые множества**: максимальные / максимальные по мощности
- **Вершинное покрытие**: минимальное по теореме Кёнига
- **Доминирующие множества**: минимальное доминирующее множество Q6
- **Хроматическое число**: раскраска, проверка k-раскрашиваемости
- **Гамильтоновы пути/циклы**: поиск, код Грея как частный случай
- **Спектральный анализ**: собственные значения матрицы смежности
- **Изоморфизм подграфов**: проверка для малых подграфов

## Спектр Q6

```
λ_k = 6 − 2k,  кратность C(6, k),  k = 0..6
  λ_0 = 6  (кратность 1)
  λ_1 = 4  (кратность 6)
  λ_2 = 2  (кратность 15)
  λ_3 = 0  (кратность 20)
  λ_4 = −2 (кратность 15)
  λ_5 = −4 (кратность 6)
  λ_6 = −6 (кратность 1)
```
