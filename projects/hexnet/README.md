# hexnet — Q6 как коммуникационная сеть

Гиперкубическая сеть Q6: 64 узла-процессора, маршрутизация, широковещание,
устойчивость к отказам, gossip-протоколы.

## Статус

Реализован

## Характеристики сети Q6

```
Узлы:            64 (= 2⁶)
Рёбра:           192 (= 64 × 6 / 2)
Степень:         6 (у каждого узла ровно 6 соседей)
Диаметр:         6 (максимальный путь)
Ширина бисекции: 32 (разбиение по одному биту)
Узловая связность: 6 (нужно 6 отказов для изоляции)
Гамильтонов:     да (содержит гамильтонов цикл)
```

## Ключевые возможности

- **E-cube маршрутизация**: детерминированная, длина = d(src, dst)
- **Адаптивная маршрутизация**: объезд отказавших узлов
- **BFS-широковещание**: рассылка от корня за 6 шагов
- **Gossip (all-to-all)**: O(log n) = 6 раундов
- **Устойчивость**: моделирование отказов узлов/рёбер
- **Балансировка нагрузки**: равномерное распределение задач
