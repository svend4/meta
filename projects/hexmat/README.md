# hexmat — Линейная алгебра над GF(2) на Q6

(GF(2))⁶ как 6-мерное векторное пространство: матрицы, ранг, ядро,
обратная матрица, группа GL(6,2).

## Статус

Реализован

## Концепция

```
Вектор:  6-битное число 0..63, bit k = компонента k
Матрица: список 6 строк, каждая rᵢ — 6-битное число
         M[i][j] = (M[i] >> j) & 1

Умножение Mv: (Mv)[i] = popcount(M[i] & v) mod 2
```

## Ключевые возможности

- **Умножение матриц**: C = A·B над GF(2)
- **Транспонирование, сложение** (XOR по строкам)
- **Ранг**: элиминация Гаусса над GF(2)
- **RREF**: reduced row echelon form
- **Ядро** (null space) и **образ** (column space)
- **Обратная матрица**: для элементов GL(6,2)
- **Определитель**: 0 или 1

## Специальные матрицы

- **GL(6,2)**: |GL(6,2)| = 20 158 709 760 = (2⁶−1)(2⁶−2)···(2⁶−32)
- Матрица перестановки, симплектическая матрица J
- **Матрица Адамара**: H[u][v] = ⟨u,v⟩ mod 2 = popcount(u&v) mod 2
