# hexca — Клеточный автомат на основе Q6

Клеточный автомат где каждая клетка — гексаграмма (64 состояния).
Аналог игры «Жизнь» Конвея, но с несравнимо богатым пространством правил.

## Статус

Идея → прототип

## Концепция

```
Обычный CA (Вольфрам):  2 состояния  → 256 возможных правил
Игра «Жизнь»:           2 состояния  → правило B3/S23
HexCA:                  64 состояния → астрономическое пространство правил
```

Правило задаётся: текущая гексаграмма клетки + значения 4 соседей → новая гексаграмма.
Переход должен быть соседом в Q6 (изменение 1 черты) — это ограничение сохраняет «плавность».

## Математическая база

- Пространство состояний = Q6
- Локальность: правило зависит только от ближайших соседей
- Обратимость: часть правил обратима (консервативный CA)
- Связь с кодами Хэмминга: можно строить CA с коррекцией ошибок

## Стек
