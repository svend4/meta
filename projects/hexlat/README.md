# hexlat — Булева решётка B₆ на Q6

Q6 = (Z₂)⁶ как булева решётка B₆: частичный порядок по включению битов,
операции meet/join, функции Мёбиуса, цепи и антицепи.

## Статус

Реализован

## Концепция

```
Булева решётка B₆:
  Элементы:       64 подмножества {0..5}
  Порядок:        x ≤ y ⟺ (x & y) == x  (включение)
  Meet:           x ∧ y = x & y  (пересечение)
  Join:           x ∨ y = x | y  (объединение)
  Дополнение:     ~x & 63

  Минимум:  0    (пустое множество)
  Максимум: 63   (полное множество)
  Высота:   6    (длина максимальной цепи)
```

## Уровневая структура

```
Ранг 0: C(6,0)=1   — {∅}
Ранг 1: C(6,1)=6   — одноэлементные
Ранг 2: C(6,2)=15  — двухэлементные
Ранг 3: C(6,3)=20  — трёхэлементные
Ранг 4: C(6,4)=15  — четырёхэлементные
Ранг 5: C(6,5)=6   — пятиэлементные
Ранг 6: C(6,6)=1   — {0..5}
```

## Ключевые возможности

- **Диаграмма Хассе**: представление частичного порядка
- **Функция Мёбиуса**: μ(x, y) = (−1)^{|y|−|x|} для B₆
- **Антицепи** (antichains): максимальная = 20 элементов (ранг 3, теорема Дилуорта)
- **Цепи**: максимальная длина = 7 (высота + 1)
- **Зета/Мёбиус преобразование**: быстрое суммирование по решётке
- **Интервалы**: [x, y] = {z : x ≤ z ≤ y}
