# hexgeom — Метрическая геометрия на Q6

Q6 = (Z₂)⁶ как метрическое пространство с расстоянием Хэмминга.
Шары, диаграммы Вороного, геодезики, упаковка и покрытие.

## Статус

Реализован

## Концепция

```
Метрика: d(a, b) = popcount(a ⊕ b)  (расстояние Хэмминга)
Шар:     B(c, r) = {h : d(h, c) ≤ r}
Сфера:   S(c, r) = {h : d(h, c) = r}

Размеры: |B(c, r)| = Σ_{k=0}^{r} C(6, k)
  r=0: 1,  r=1: 7,  r=2: 22,  r=3: 42,  r=4: 57,  r=5: 63,  r=6: 64
```

## Ключевые возможности

- **Диаграмма Вороного**: разбиение Q6 по ближайшим центрам
- **Граф Делоне**: двойственный к Вороного
- **Геодезики**: {h : d(u,h)+d(h,v)=d(u,v)} — множество точек на кратчайшем пути
- **Медиана Штейнера**: минимизация суммы расстояний до набора точек
- **Упаковка (packing)**: максимум непересекающихся шаров радиуса r
- **Покрытие (covering)**: минимум шаров, покрывающих весь Q6
- **Изоперметрическое неравенство**: граница вершинного расширения

## Математическая база

- Метрическое пространство (Q6, Хэмминг) — нормированный вектор пространство над GF(2)
- Граница Хэмминга: A(6, d) ≤ 64 / |B(0, ⌊(d-1)/2⌋)|
- Граница Синглтона: A(6, d) ≤ 2^(6-d+1)
