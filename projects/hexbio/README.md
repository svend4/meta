# hexbio — Биоинформатика: генетический код на Q6

64 кодона РНК = 64 гексаграммы. Точечные мутации = переходы по графу Q6.

## Статус

Реализован

## Концепция

```
Кодирование нуклеотидов (2 бита):
  A = 00,  C = 01,  G = 10,  U = 11

Кодон XYZ → 6-битное число: [5:4]=X, [3:2]=Y, [1:0]=Z

Точечная мутация (замена нуклеотида) = XOR двух соседних битов
Граф мутаций ≈ подграф Q6
```

## Математическая база

- 64 кодона ↔ 64 элемента (Z₂)⁶
- Кодирование нуклеотидов: 2 бита на позицию
- Мутационные расстояния: метрика Хэмминга по нуклеотидам
- Генетический код: отображение 64 кодона → 20 аминокислот + стоп
- Вырожденность (degeneracy): ~3 кодона на аминокислоту в среднем
- Мутационные пути: кратчайшие пути в графе мутаций

## Ключевые функции

```python
codon_to_int("AUG")   # Метионин (старт-кодон) → 32
int_to_codon(32)      # → "AUG"
genetic_code(n)       # кодон → аминокислота
point_mutations(n)    # все одиночные мутации кодона
```
