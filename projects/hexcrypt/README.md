# hexcrypt — Криптографические примитивы на Q6

Учебная реализация симметричной криптографии на пространстве Q6 = (Z₂)⁶.
S-блоки, потоковые шифры, сети Фейстеля, дифференциальный и линейный анализ.

## Статус

Реализован

## Концепция

```
Q6 — учебная платформа классической симметричной криптографии:
  • S-блок:          биективная подстановка {0..63} → {0..63}
  • Поточный шифр:   LFSR с состоянием ∈ Q6
  • Сеть Фейстеля:   разбиение 3|3 бита
  • DDT / LAT:       дифференциальные и линейные характеристики
```

## Ключевые возможности

- **S-блок**: нелинейность, DDT (таблица разностных распределений), LAT
- **SAC** (Strict Avalanche Criterion): проверка лавинного эффекта
- **Алгебраическая степень**: максимальная степень компонентных функций
- **Поточный шифр**: LFSR-подобный с 64-состояниями
- **Сеть Фейстеля**: 3+3 бита, обратимая конструкция
- **Криптоанализ**: утилиты для линейного и дифференциального анализа

## Математическая база

- Нелинейность: NL(S) = (64 − max|LAT|) / 2
- Оптимальный S-блок: NL = 24 (для 6→6 подстановки)
- Дифференциальная равномерность: max(DDT) = 4 → APN, = 2 → идеал
- Аффинные подстановки: A(x) = Mx ⊕ v над GF(2)^6
