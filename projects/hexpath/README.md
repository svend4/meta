# hexpath — Абстрактная стратегическая игра

Двухигровая стратегия на графе Q6. Игроки навигируют по 64 гексаграммам,
меняя одну черту за ход. Первый кто достигает цели или блокирует соперника — побеждает.

## Статус

Идея → прототип

## Правила (базовый вариант)

1. Доска = граф Q6 (64 узла, 192 ребра)
2. Каждый игрок выбирает стартовую гексаграмму
3. Ход = переход по одному ребру (изменение 1 черты)
4. Цель = достичь заданной целевой гексаграммы ИЛИ поставить соперника в позицию без ходов
5. Вариант «Захват»: узлы захватываются при посещении, соперник не может ходить на захваченные

## Почему это интересно математически

- Диаметр Q6 = 6 (из любой позиции цель достижима за ≤6 ходов)
- Идеальная информация (как шахматы/го)
- Доказуемо симметричная начальная позиция из-за свойств группы (Z₂)⁶
- Возможен полный анализ игры (64 узла — небольшое дерево решений)

## Стек

Python 3.10+ (логика игры + CLI). Опционально: pygame для GUI.
