# hexring — Булевы функции и алгебраические структуры на Q6

Булевы функции f: (Z₂)⁶ → {0,1}, АНФ, WHT, нелинейность, bent-функции,
корреляционный иммунитет, коды Рида–Маллера.

## Статус

Реализован

## Концепция

```
Булева функция f: {0,1}⁶ → {0,1}
Таблица истинности: TT ∈ {0,1}⁶⁴ (64 бита)

Алгебраическая нормальная форма (АНФ):
  f(x) = Σ_{I⊆{0..5}} a_I · x^I   (мономы над GF(2))

Кольцевая структура: (функции, XOR, AND) — булево кольцо
```

## Ключевые характеристики

| Характеристика | Описание |
|----------------|----------|
| WHT / Уолш-спектр | Ŵ(u) = Σ (−1)^{f(x)+u·x} |
| Нелинейность | nl(f) = (64 − max|Ŵ|) / 2 |
| Bent-функция | nl(f) = 28, |Ŵ(u)| = 8 ∀u (максимум для n=6) |
| Корреляционный иммунитет | CI(t): Ŵ(u)=0 для 0<weight(u)≤t |
| Алгебраическая степень | max степень монома в АНФ |

## Коды Рида–Маллера RM(r, 6)

```
RM(0, 6): константы          — длина 64, размерность 1,  d=64
RM(1, 6): аффинные функции   — длина 64, размерность 7,  d=32
RM(2, 6): квадратичные       — длина 64, размерность 22, d=16
RM(3, 6): кубические         — длина 64, размерность 42, d=8
```
