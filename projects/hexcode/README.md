# hexcode — Двоичные линейные коды в Q6

Теория кодирования на пространстве Q6 = (Z₂)⁶. Линейные коды длины 6,
синдромное декодирование, стандартные конструкции.

## Статус

Реализован

## Концепция

```
Q6 = (Z₂)⁶ — пространство кодовых слов длины n=6.
Линейный [n,k,d]-код = подпространство размерности k.
Синдром s = H·e (ошибка e), декодирование = таблица синдромов.
```

## Стандартные коды

| Код | Параметры | Описание |
|-----|-----------|----------|
| `repetition_code()` | [6,1,6] | 1 бит, повторённый 6 раз |
| `parity_check_code()` | [6,5,2] | 5 инф. бит + бит чётности |
| `dual_parity_code()` | [6,2,4] | минимальный дистанционный |
| `shortened_hamming()` | [6,3,4] | укороченный код Хэмминга |
| `even_weight_code()` | [6,5,2] | слова чётного веса |
| `all_ones_code()` | [6,1,6] | {000000, 111111} |

## Математическая база

- Порождающая матрица G (k×6), проверочная матрица H ((n-k)×6)
- Кодовое расстояние d = min вес ненулевого кодового слова
- Синдромное декодирование: ближайший сосед за O(2^(n-k))
- Покрывающий радиус: max расстояние до ближайшего кодового слова
- Совершенный код: шары Хэмминга покрывают всё (Z₂)⁶ без перекрытий
