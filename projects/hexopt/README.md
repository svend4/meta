# hexopt — Оптимизация на Q6

Алгоритмы дискретной оптимизации на пространстве Q6 = (Z₂)⁶:
локальный поиск, имитация отжига, генетический алгоритм, поиск с запретами.

## Статус

Реализован

## Концепция

```
Два класса задач:

  [Одна точка]  h* = argmax f(h),   h ∈ Q6
    Окрестность h: 6 соседей (расстояние Хэмминга 1)

  [Подмножество] S* = argmin/argmax g(S),  S ⊆ Q6
    Представление: 64-битная маска
    Окрестность: добавить/убрать одну гексаграмму
```

## Реализованные алгоритмы

| Алгоритм | Описание |
|----------|----------|
| `local_search` | Подъём по холму + случайные перезапуски |
| `simulated_annealing` | Имитация отжига (критерий Метрополиса) |
| `genetic_algorithm` | ГА с однородным кроссовером и мутацией |
| `tabu_search` | Поиск с запретами (запрет недавних состояний) |

## Задачи на Q6

```python
weighted_yang(w)      # h: максимизировать взвешенный счёт ян
min_dominating_set()  # S: минимальное доминирующее множество (оптимум = 8)
max_hamming_spread(k) # S: k точек с максимальным попарным расстоянием
min_covering_code(t)  # S: минимальный код с покрывающим радиусом ≤ t
```
