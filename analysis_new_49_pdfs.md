### 3.1 `372` — Группа сечений R6 (Closure of Cross-Ratio)

**Формулы:**
```
Двойное отношение 4 точек:
w = (1234) = (13)·(42) / (32)·(14)

6 значений при перестановках = группа R6:
r0 = w           r1 = 1/w        r2 = 1 - w
r3 = 1/(1-w)     r4 = (w-1)/w    r5 = w/(w-1)

R6 ≅ S3 (симметрическая группа степени 3)

Четырёхгруппа Клейна V4 = {w, 1/w* ... } (4 пары точек с одним w):
V4 ⊗ S3 = S4  (симметрическая группа степени 4)

Тождества:
∏r_i = 1    ΣR_i = 3

Правило умножения: r_i ∘ r_j = r_i(r_j(w))  (подстановка)
```

**Применение:** проективная геометрия, компьютерное зрение (homography), криптография (дробно-линейные преобразования), образование.

### 3.2 `4c5` — Уравнение X^Y = Y^X

**Формулы:**
```
Параметрическое решение (t = Y/X):
X(t) = t^(1/(t-1))
Y(t) = t^(t/(t-1))

Симметрия: X(1/t) = Y(t)  — при замене t→1/t переменные меняются местами

Предел: lim_{t→1} X(t) = e  (исключение: x=e имеет только тривиальное решение)

Теорема об исключении: для всех X > 1, X ≠ e, существует Y ≠ X: X^Y = Y^X.
При X = e единственное решение Y = e.

«Золотые уравнения» (единственный корень φ):
  x^(1/(x-1)) = (1+1/x)^x           → x = φ
  x^(x/(x-1)) = (1+1/x)^(x+1)       → x = φ

«Золотое» решение при t = φ: Y = φ·X
```

### 3.3 `708` — Уникальные группы

**Формулы:**
```
Q(n) = число различных групп порядка n

Определение уникальной группы: G(n) уникальна, если Q(n) = 1 и n не простое
Первая уникальная группа: G(15) = G(3) ⊗ G(5)

Классификация нечётных чисел:
U = {1,7,13,19,...}: u_n = 6n - 5  (≡1 mod 6)
V = {3,9,15,21,...}                (≡3 mod 6, кратные 3)
W = {5,11,17,23,...}: w_n = 6n - 1 (≡5 mod 6)

Таблица умножения классов:
[U×U]=U, [W×W]=U, [U×W]=W, [U×V]=[W×V]=[V×V]=V

Гипотеза: группа порядка из класса V никогда не уникальна

Формула хорд V (n точек, k непересекающихся хорд):
V = (1/k!) · ∏_{i=1}^{k} C²_{n-2(i-1)}
где C²_m = m(m-1)/2

Пример: n=6, k=3: V = (1/6)·C²₆·C²₄·C²₂ = 15

35 уникальных порядков до 300:
15, 33, 35, 51, 65, 69, 77, 85, 87, 91, 95, 115, 119, 123, 133, 141, 143, 145,
159, 177, 185, 187, 209, 213, 217, 235, 247, 249, 255, 259, 265, 267, 287, 295, 299
```

### 3.4 `dbf` — Формула паркета (Generalized Buffon)

**Формулы:**
```
Классический Бюффон: W = (2L)/(π·a)  [игла длиной L, доски шириной a]

Обобщение для прямоугольного паркета (сторона a × b):
W = (L/π) · (1/a + 1/b) = L·U/(π·F)
  где U = периметр клетки = 2(a+b), F = площадь клетки = a·b

Частный случай — золотой прямоугольник a × a·φ, карандаш L = a·e/2:
W = φ·e / π  ≈ 1.4008...
(связывает φ, e, π в одной формуле)

Параметрическая форма (λ = b/a):
W = (2L / (π·a)) · (1 + 1/λ)

При λ→∞ (одна система досок): W = 2L/(π·a)  — формула Бюффона ✓
```

### 3.5 `677` — Топология многогранников (перебор)

```

(C_B=4, C_Γ=4): Γ=B=k ∀k ∈ N
(C_B=6, C_Γ=3): Γ=2B
(C_B=3, C_Γ=6): 2Γ=B
```

### 3.7 `a08` — Скрытые параметры эллипса

**Формулы:**
```
Параметры эллипса с полуосями a, b (b < a):

Фокальный параметр:  p₁ = b²/a  (радиус кривизны при вершинах ±a)
Радиальный параметр: p₂ = a²/b  (радиус описанной окружности)

Фундаментальное тождество: a·b = c·p  (p = ab/c — осевой параметр)
Перемножение: p₁·p₂ = a²b²/(ab) = ab  → a·b = c·p = p₁·p₂ / c... (8)

Уравнение «линии катастроф» (экидистанты на расстоянии q=p₁ от эллипса):
q_кр = p₁ = b²/a  — переход к особой кривой

Горизонтальное смещение (2):
(x₁-x₂)² = q²·b²x₁² / (a⁴ - a²x₁² + b²x₁²)

Уравнение эквидистанты (3):
y₂ = √(a²-x₁²)·(b/a - qa/√(a⁴ - x₁²c²))

Задача вписанных окружностей (четыре пересечения):
(a²-b²)y² + 2a²(b-R)y + R²(a²+b²) - 2Rba² = 0  →  R = a²/b = p₂
```

### 3.8 `c60` — Развёртки куба (11 штук)

**Алгоритм:**
```
Куб: 6 граней, 12 рёбер. Для развёртки нужно разрезать ровно 7 рёбер.
Общее число способов: V = C¹²₇ = 792

Граф: 6 вершин (грани), ребро ↔ смежные грани.
Развёртка = остов графа (spanning tree) из 5 рёбер.

V₂ = 324 + 60 + 24 = 408

Проверка: V₁ + V₂ = 384 + 408 = 792 = V ✓
→ Ровно 11 различных развёрток куба (6 симметричных + 5 асимметричных)
```

### 3.9 `edf` — Математическая модель машины времени (Тор)

**Формулы:**
```
Параметризация тороида (намотка с коэффициентом ω):
ρ⃗ = (1 + cos(φ/ω))·e⃗ + sin(φ/ω)·k⃗

Группа порядка 6 (алгебраическое представление):
x₁=ω, x₂=-(1+ω)/ω, x₃=-1/(1+ω)
t₁=1/ω, t₂=-ω/(1+ω), t₃=-(1+ω)

Операция: x_i ∘ t_j = подставить t_j вместо ω в x_i и упростить
Пример: x₃ ∘ t₂ = -t₂/(1+t₂) = t₁

Геодезические уравнения на торе (1):
d²θ/dt² - sin²θ·sin(2θ)·(dφ/dt)² = 0
d²φ/dt² + 4·ctg(θ)·(dθ/dt)(dφ/dt) = 0

Теорема 1: сфера радиуса R=√(r₁²+r₂²) делит тор ровно пополам:
V₁ = π²r₁r₂² = V/2  (V = 2π²r₁r₂²)

Теорема 2: вращение хорды тора даёт тело объёма = шар радиуса r₂:
V = (4/3)πr₂³  (не зависит от r₁!)
```

---

### 3.10 `8f6` — Теорема о трёх элементарных пучках

**Результат:**
```
Система обозначений: три пучка с центрами A₁, A₂, A₃ на одной прямой.
Точки пересечений: a^(ij)_(ba) = правая линия пучка i ∩ левая линия пучка j

Теорема (объединяет Декарга и Паскаля):
1. Координированные диагонали попарных сечений 3 пучков → одна точка (Декарг)
2. Пересечения противоположных сторон «мерцающих» шестиугольников → одна прямая (Паскаль)

Из одной теоремы выводятся:
- Теорема Декарга о перспективных треугольниках
- Теорема Паскаля о вписанном шестиугольнике
```

### 3.11 `e3d` — Рогатый полиэдр

**Формулы:**
```
Ограничение на параметры сети (a=1):
2b³ - 2b²a - ba²(1+2√3) + a³(1+√3) = 0

При a=1:
(b-1)(2b²-1-√3) - b√3 = 0

Численное решение: b/a ≈ 1.81509...
(корень кубического уравнения — нельзя построить циркулем и линейкой!)

Симметрия: ось через середины сторон DE, GH, AB
```

---

### 3.12 `d4b` — Теорема Прасолова об арбело

**Формула:**
```
Для остроугольного треугольника со сторонами a, b, c:
Построить окружности на каждой стороне как диаметре.

S_внешн - S_внутр = 2·S_треугольника

где S_внешн = сумма площадей сегментов окружностей вне треугольника
    S_внутр = площадь криволинейного треугольника внутри
    S_треугольника = площадь исходного треугольника
```
