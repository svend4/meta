## Статус извлечения текста

```
Попытки извлечения:
  strings          — только бинарные символы (PDF с CIDFont / embedded font)
  zlib decompress  — структура PDF (BT/ET блоки, Pagination артефакты)
  pypdf            — установлен, но _cffi_backend недоступен в среде
  pdftotext        — не установлен (требует poppler-utils)

Вывод: PDF-файлы используют встроенные шрифты с нестандартной
кодировкой (Type0/CIDFont). Для извлечения текста необходим
инструмент с поддержкой ToUnicode CMap.
```

### Как читать статьи

```bash
# Установить poppler-utils (Ubuntu/Debian):
sudo apt-get install poppler-utils

# Затем извлечь текст:
for f in *.pdf; do pdftotext "$f" "${f%.pdf}.txt"; done

# Или через Python после установки:
pip install pymupdf
python3 -c "
import fitz, os
for fname in sorted(os.listdir('.')):
    if fname.endswith('.pdf'):
        doc = fitz.open(fname)
        text = ''.join(p.get_text() for p in doc)
        print(fname, ':', text[:100])
"
```

## Связь с проектами Q6

### Дифференциальная геометрия → hexgeom

Метрические свойства Q6 = (Z₂)⁶:
- Шары Хэмминга ≈ дискретные аналоги шаров в риманновом многообразии
- Геодезики (кратчайшие пути) = минимальные последовательности битовых флипов
- «Касательные сферы» → перекрывающиеся шары Хэмминга

### Непериодические мозаики → hexca, hexgraph

- Клеточный автомат hexCA: непериодические орбиты правил
- Подграфы Q6: апериодические паттерны (аналог мозаик Пенроуза в дискретном пространстве)

### Тороидальные структуры → hexnet, hexca

- Q6 с периодическими граничными условиями = тор (Z₂)⁶
- hexnet: маршрутизация на торе vs. граф Q6
- hexCA с периодическими границами = CA на торе

### Теория сфер → hexgeom, hexcode

- Упаковка шаров = коды с минимальным расстоянием
- Граница Хэмминга: A(6,d) ≤ 64/|B(0,⌊(d-1)/2⌋)|
- Совершенные коды = идеальные упаковки шаров
