# Математические основы графа Q6

## Определение

**Q6** — 6-мерный гиперкуб. Граф где:
- **Вершины**: все 6-битные числа {0, 1}⁶ = 64 узла
- **Рёбра**: два узла соединены ↔ они отличаются ровно в 1 бите

Это граф Кэли группы **(Z₂)⁶** с порождающим множеством
{e₀, e₁, e₂, e₃, e₄, e₅} где eᵢ = переворот бита i.

## Ключевые свойства

| Свойство | Значение |
|---|---|
| Число вершин | 64 |
| Число рёбер | 192 |
| Степень каждой вершины | 6 |
| Диаметр | 6 |
| Радиус | 6 |
| Регулярность | 6-регулярный |
| Двудольность | да (чётные/нечётные числа ян-линий) |
| Гамильтонов путь | существует (= код Грея) |
| Хроматическое число | 2 |

## Расстояние Хэмминга

Расстояние между двумя вершинами a и b в Q6 = число бит,
в которых они отличаются:

```
d(a, b) = |{i : aᵢ ≠ bᵢ}| = popcount(a XOR b)
```

Это и есть метрика Хэмминга. Кратчайший путь между a и b
имеет длину d(a, b) и достигается переворотом отличающихся бит в любом порядке.

## Код Грея = Гамильтонов путь по Q6

Стандартный рефлексивный код Грея:

```
i = 0: 000000
i = 1: 000001
i = 2: 000011
i = 3: 000010
...
i = n: n XOR (n >> 1)
```

Посещает все 64 вершины, каждый раз меняя ровно 1 бит.
Используется в АЦП, энкодерах, адресации памяти.

## Связь с кодами Хэмминга

[6,1]-код Хэмминга: добавление 1 бита чётности к 6-битному слову.
Граф Q6 = пространство кодовых слов этого кода.

Обобщение: коды Хэмминга [2ⁿ-1, 2ⁿ-n-1] живут в гиперкубах Qₙ.

## Связь с квантовыми вычислениями

6 порождающих операций Q6 = 6 ворот Паули-X на 6-кубитном регистре:

```
X₀|b₅b₄b₃b₂b₁b₀⟩ = |b₅b₄b₃b₂b₁(1-b₀)⟩
X₁|b₅b₄b₃b₂b₁b₀⟩ = |b₅b₄b₃b₂(1-b₁)b₀⟩
...
```

Стабилизаторные коды (основа квантовой коррекции ошибок) используют
именно структуру гиперкуба над (Z₂)ⁿ.

## Связь с картами Карно

Карта Карно — это способ нарисовать часть гиперкуба:

```
Карно 2×2 = Q2 (4 вершины)
Карно 2×4 = Q3 (8 вершин)
Карно 4×4 = Q4 (16 вершин, стандарт)
Карно 4×8 = Q5 (32 вершины, редко)
Q6 = следующий уровень (64 вершины, этот проект)
```

В карте Карно минимизация булевой функции = нахождение
максимальных кубов в подграфе Q6 из единиц функции.

## Связь с синтезом автоматов

Любой конечный автомат с не более чем 64 состояниями,
где переходы меняют одно свойство за раз, можно вложить в Q6.

Это применяется в:
- сетевых протоколах (TCP: 11 состояний → подграф Q6)
- компиляторах (состояния лексера)
- встроенных системах (state machine с булевыми флагами)
